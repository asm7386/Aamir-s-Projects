# Lending Club Loan Analysis

This project analyzes Lending Club loan data to explore patterns in lending behavior and loan performance. The goal is to clean the dataset, analyze important variables, and identify factors that may influence loan outcomes.

## Objectives

- Clean and preprocess the loan dataset
- Perform exploratory data analysis (EDA)
- Analyze key financial variables related to loans
- Visualize trends and patterns in the data

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Overview

The notebook performs data preprocessing, analysis, and visualization to better understand loan characteristics and lending patterns within the dataset.
