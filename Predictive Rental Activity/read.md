# Bike Rental Feature Engineering & Prediction

This project analyzes a bike rental dataset to explore demand patterns and build predictive features for rental activity. The analysis focuses on cleaning the dataset, creating new features, and examining relationships that influence bike rental demand.

## Objectives

- Clean and preprocess the bike rental dataset
- Perform exploratory data analysis (EDA)
- Create engineered features for improved modeling
- Analyze factors that influence rental demand

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Overview

The notebook performs data preparation, feature engineering, and exploratory analysis to better understand patterns in bike rental usage and prepare the data for predictive modeling.
