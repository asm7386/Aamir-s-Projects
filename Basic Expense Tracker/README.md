# Expense Tracker

This project is a simple expense tracking application built using Python. The goal of the project is to record, organize, and analyze personal expenses to better understand spending habits.

## Features

- Record expenses with categories and amounts
- Track total spending
- Organize expenses by type
- Basic data analysis and visualization

## Technologies Used

- Python
- Pandas
- Jupyter Notebook
- Matplotlib

## Project Structure
Expense-Tracker
│
├── ExpenseTracker.ipynb
└── README.md


The notebook contains the code used to load expense data, perform analysis, and generate visualizations.

## Purpose

This project demonstrates basic data analysis techniques including data organization, aggregation, and visualization using Python.
