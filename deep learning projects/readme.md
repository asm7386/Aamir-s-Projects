# Deep Learning Capstone Project

This project explores deep learning techniques across multiple data domains including tabular data, images, and text. The goal is to design, train, and evaluate neural network models to solve real-world problems such as customer churn prediction, face mask detection, sentiment analysis, and image denoising.

These tasks demonstrate practical experience with neural networks, transfer learning, natural language processing, and autoencoders using modern deep learning frameworks.

---

# Project Tasks

## Task 1: Customer Churn Prediction

This task focuses on predicting whether a bank customer will churn using a structured banking dataset.

Key steps included:

- Preprocessing the dataset by removing unnecessary features
- Encoding categorical variables
- Splitting the data into training and test sets
- Building a Sequential Artificial Neural Network using Keras
- Training the model using appropriate activation functions, optimizer, and loss function
- Evaluating model performance using accuracy and confusion matrix metrics

This task demonstrates how deep learning models can be applied to tabular financial datasets.

---

## Task 2: Face Mask Detection (Transfer Learning)

In this task, transfer learning techniques were used to detect whether people are wearing face masks in images.

Key steps included:

- Loading and preprocessing image datasets
- Preparing training, validation, and test datasets
- Implementing transfer learning models with pooling, dropout, and softmax layers
- Training and optimizing the model using callbacks
- Evaluating model performance and visualizing predictions

This task demonstrates the use of pretrained convolutional neural networks for computer vision problems.

---

## Task 3: Customer Review Sentiment Classification

This task applies deep learning techniques to classify customer product reviews based on sentiment.

Key steps included:

- Generating a binary target variable from review ratings
- Tokenizing and padding text sequences
- Splitting the dataset into training and test sets
- Designing a hybrid CNN-LSTM model
- Training the model using embedding, convolutional, and LSTM layers
- Evaluating the model using accuracy and loss metrics

This task demonstrates the use of neural networks for natural language processing.

---

## Task 4: Image Denoising with Autoencoders

The final task focuses on image reconstruction using autoencoders.

Key steps included:

- Processing image datasets and converting them into arrays
- Adding Gaussian noise to simulate corrupted images
- Normalizing pixel values
- Designing a convolutional autoencoder architecture
- Training the model to reconstruct clean images
- Evaluating results by comparing noisy and reconstructed images

This task demonstrates the use of unsupervised deep learning for image restoration.
